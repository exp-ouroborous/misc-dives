# Dives

Each folder here is a small, self-contained learning dive.

## Current Dives

- [Regex Tester](./regex-tester/) - Python `re` matching, groups, flags, and substitution in the browser.
- [JSON Pretty Printer](./json-pretty-printer/) - Python `json` formatting, minifying, sorting, and validation.
- [Text Diff](./text-diff/) - Python `difflib` unified diffs for two text blocks.
- [Markdown To HTML](./markdown-to-html/) - real Python `markdown` package conversion through Pyodide and `micropip`.
